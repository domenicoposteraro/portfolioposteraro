Nmap

Cos'è Nmap?

Nmap (Network Mapper) è uno strumento di scansione di rete ampiamente utilizzato che consente di rilevare host e servizi all'interno di una rete. È in grado di determinare quali porte sono aperte, quale software e versione è in esecuzione su un sistema, e può anche fornire informazioni sulla topologia di rete.

Funzioni principali di Nmap

  • Scansione delle porte di un host o di un'intera rete
  • Rilevamento degli host attivi nella rete
  • Rilevamento dei servizi e delle versioni in esecuzione su un host
  • Rilevamento delle vulnerabilità e delle configurazioni di sicurezza
  • Mapping della topologia di rete
  • Generazione di report dettagliati delle scansioni

Comandi principali di Nmap

Scansione delle porte di un host:

nmap

Scansione delle porte di una rete:

nmap

Scansione delle porte specifiche:

nmap -p

Scansione in modalità silenziosa:

nmap -sS

Rilevamento dei sistemi operativi:

nmap -O

Scansione usando uno script NSE:

nmap --script=
